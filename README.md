# Traveloop ✈️  
### Odoo x KAHE Hackathon Project

Traveloop is a modern cinematic travel planning web application designed to simplify and enhance trip management.  
Built for the **Odoo x KAHE Hackathon**, the platform provides an immersive and visually engaging experience for users to plan, organize, and manage their journeys efficiently.

---

## 🌍 Project Overview

Traveloop helps travelers:

- Create and manage trips
- Build detailed itineraries
- Track travel budgets
- Organize packing checklists
- Discover destinations and activities
- Share travel plans with others

The project combines modern UI/UX design with interactive animations to deliver a premium travel-planning experience.
